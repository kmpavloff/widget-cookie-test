🔬 Cookie SameSite Comparison Test Strict Lax Partitioned

Что тестируется: сайт example.localhost выставляет три куки:
  🔴 session_id — SameSite=Strict — не отправляется в cross-site iframe вообще
  🟡 lax_id — SameSite=Lax — отправляется при top-level GET, но не при POST и не в iframe
  🟢 partitioned_id — SameSite=None; Partitioned (CHIPS) — отправляется всегда, партиционируется по origin

Ожидаемый результат:
  🔴 Strict: в cross-site iframe не отправляется → сервер каждый раз создаёт новую
  🟡 Lax: в cross-site iframe не отправляется (только top-level GET)
  🟢 Partitioned: каждый top-frame origin получает свою куку, внутри одного origin — сохраняется
Действия:
🖥️ Прямой доступ example.localhost напрямую
📦 Через site-a iframe на site-a.localhost
📦 Через site-b iframe на site-b.localhost
